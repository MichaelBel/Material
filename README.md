## Requirements

* Android 5.0
* Android Studio 2.1.2

## Import
Project build.gradle
```gradle
allprojects {
    repositories {
        ...
        maven { url "https://jitpack.io" }
    }
}
```

Module build.gradle
```gradle
dependencies {
    compile 'com.github.michaelbel:material:v1.0.0'
}
```
